# Telemost Transcriber

Локальное приложение для расшифровки записей Яндекс Телемоста.

Программа принимает видео или аудио встречи, извлекает звук, распознает речь локально через Whisper и формирует протокол через OpenAI GPT.

## Что умеет

- Загружать видео или аудио записи встреч.
- Извлекать аудио через встроенный `ffmpeg`.
- Распознавать речь локально через `faster-whisper`.
- Работать с моделями Whisper `small` и `medium`.
- Делать итоговый протокол через GPT.
- Показывать время обработки.
- Сохранять историю обработок.
- Выдавать результаты в форматах `TXT`, `Markdown`, `DOCX`.

## Что получается на выходе

После обработки доступны:

- полный транскрипт с таймкодами;
- краткое резюме встречи;
- договоренности;
- задачи;
- ответственные;
- сроки;
- вопросы для уточнения;
- Word-файл с протоколом.

## Быстрый старт

1. Распакуйте архив в любую удобную папку.
2. Запустите `Установить Telemost Transcriber.vbs`.
3. Дождитесь завершения установки в окне установщика.
4. На рабочем столе появится ярлык `Telemost Transcriber`.
5. Запустите ярлык.
6. В браузере откроется локальный сайт.
7. Введите OpenAI API key в блоке настроек на сайте.
8. Загрузите запись встречи и запустите обработку.

Python отдельно устанавливать не нужно. Он уже находится внутри архива в папке `python`.

## Первый запуск

При первом запуске нужно:

1. Ввести OpenAI API key на сайте.
2. Выбрать файл записи.
3. Выбрать модель Whisper.
4. Нажать `Запустить обработку`.

При первой обработке выбранная модель Whisper будет скачана автоматически. Для этого нужен интернет.

Для первого теста рекомендуется выбрать `small`.

## Как запускать после установки

После установки используйте ярлык на рабочем столе:

```text
Telemost Transcriber
```

Ярлык запускает локальный сервер без черного консольного окна и открывает браузер.

Если ярлык не появился, можно запустить приложение прямо из папки:

```text
Запустить Telemost Transcriber.vbs
```

Это запасной запуск без консольного окна. Файл `start.bat` оставлен как технический вариант для диагностики.

## Как остановить приложение

В правом верхнем углу сайта есть кнопка:

```text
Выключить приложение
```

Нажмите ее, когда закончили работу: локальный сервер остановится, и вкладку браузера можно закрыть. Для нового запуска используйте ярлык на рабочем столе.

Если приложение запущено через `start.bat`, его также можно остановить через `Ctrl+C` в консольном окне.

## Где хранятся данные

Все данные хранятся внутри папки проекта.

```text
data/uploads/        исходные видео и аудио
data/audio/          извлеченный звук
data/transcripts/    транскрипты
data/reports/        протоколы
data/jobs/           история задач
models/              скачанные модели Whisper
runtime/             установленные зависимости
.env                 локальные настройки и OpenAI API key
```

Аудио не отправляется в OpenAI. В OpenAI отправляется только текстовый транскрипт, чтобы GPT сформировал протокол.

## Как удалить программу

1. Остановите приложение кнопкой `Остановить`.
2. Удалите папку проекта.
3. Удалите ярлык `Telemost Transcriber` с рабочего стола.

Приложение не устанавливается в систему глобально. Основные файлы лежат в папке проекта.

## Настройки

Нейросеть для протокола выбирается на сайте в блоке настроек кнопкой `Изменить`.

Доступные провайдеры:

```text
OpenAI      модели gpt-4.1-mini, gpt-4.1, gpt-4o и другие
DeepSeek    модели deepseek-chat, deepseek-reasoner (работает из России)
YandexGPT   модели yandexgpt/latest и другие (нужен ID каталога Yandex Cloud)
```

В настройках выбираются провайдер, модель и ключ. Кнопка `↻` рядом с моделью
подтягивает актуальный список моделей под ваш ключ. Для YandexGPT дополнительно
указывается `ID каталога` (folder id) из консоли Yandex Cloud.

После сохранения настройки записываются в файл `.env`, например:

```text
LLM_PROVIDER=deepseek
OPENAI_API_KEY=...
DEEPSEEK_API_KEY=...
DEEPSEEK_MODEL=deepseek-chat
YANDEX_API_KEY=...
YANDEX_FOLDER_ID=...
YANDEX_MODEL=yandexgpt/latest
WHISPER_MODEL=small
```

`LLM_PROVIDER` задает активного провайдера для создания протокола.

`WHISPER_MODEL` задает модель Whisper по умолчанию. В интерфейсе можно выбрать `small` или `medium`.

## Модели Whisper

Доступны две модели:

```text
small   быстрее, подходит для первого теста
medium  точнее, но работает дольше
```

На компьютерах без NVIDIA GPU обработка идет на CPU и может занимать заметное время.

## Частые вопросы

### Нужно ли ставить Python?

Нет. Python уже находится в папке `python`.

### Нужно ли ставить ffmpeg?

Нет. Приложение использует встроенный `ffmpeg` из Python-пакета `imageio-ffmpeg`.

### Почему первая обработка долгая?

При первой обработке скачивается модель Whisper. Следующие обработки с этой моделью начнутся быстрее.

### Где взять OpenAI API key?

Ключ создается в аккаунте OpenAI Platform. Его нужно вставить в блок настроек на сайте.

### Можно ли использовать без OpenAI API key?

Транскрипт можно получить, но GPT-протокол с резюме, задачами и договоренностями не будет создан.

### Что делать, если сайт не открывается?

Проверьте:

- запускали ли вы `Установить Telemost Transcriber.vbs`;
- есть ли папка `runtime`;
- не занят ли порт `8000` (если он занят, приложение уже запущено — просто откройте `http://127.0.0.1:8000` в браузере);
- не заблокировал ли запуск антивирус или Windows SmartScreen;
- появился ли файл `launcher_error.log` в папке проекта — в нем сохраняется причина сбоя запуска.

### Что делать, если обработка зависла?

Остановите приложение кнопкой `Остановить`, запустите заново и повторите обработку. Незавершенные задачи будут помечены как прерванные.

## Безопасность

Не публикуйте свой файл `.env`.

Если вы делитесь архивом с другими людьми, делитесь исходным архивом без `.env`, без папок `data`, `models` и `runtime` с личными данными.

OpenAI API key хранится локально и не отображается в интерфейсе после сохранения.
